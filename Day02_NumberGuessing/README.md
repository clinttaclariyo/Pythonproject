# 🎯 Number Guessing Game – Day 2 Python Project

This is a fun command-line Python game where the user has to guess a number between 1 and 10. The user gets 3 chances.

---

## 📚 Concepts Used

- `input()` and `int()`
- `if`, `elif`, `else`
- `while` loop
- `break`, `continue`
- `random.randint()`
- `isdigit()` for input validation
