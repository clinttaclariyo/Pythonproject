# 🛒 Grocery List App – Day 4 Python Project

This is a simple interactive app where the user adds grocery items until they type "done". The app then prints a numbered list of items.

---

## 📚 Concepts Used
- Lists (`append()`)
- Loops (`while`, `for`)
- User input
- `enumerate()`
